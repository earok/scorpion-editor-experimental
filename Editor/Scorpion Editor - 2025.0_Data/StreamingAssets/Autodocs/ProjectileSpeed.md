### ProjectileSpeed

**Category:**
Projectile

**Syntax:**

```scorpionengine
ProjectileSpeed X=Expression Y=Expression Angle=Expression
```

**Description:**

Set the exact speed of this entity

X: The X coordinate
Y: The Y coordinate
Angle: The angle 0-359

```scorpionengine

ProjectileSpeed X=5*MyVar Y=5*MyVar Angle=180

```
