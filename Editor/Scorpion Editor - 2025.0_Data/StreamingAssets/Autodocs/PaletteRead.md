### PaletteRead

**Category:**
Palette

**Syntax:**

```scorpionengine
PaletteRead "FileName" PaletteFrom
```

**Description:**

Read a single color from a PNG into RED, GREEN and BLUE

FileName: The relative file path
PaletteFrom: The source index of the palette operation

```scorpionengine

PaletteRead "MyFile.txt" 0

```
