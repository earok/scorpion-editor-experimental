### ElseIf

**Category:**
Conditional

**Syntax:**

```scorpionengine
ElseIf Conditions
```

**Description:**

An IF statmement that only runs if the previous IF was false

Conditions: A number of expressions such as X>5 separated by AND or OR

```scorpionengine

ElseIf 5 > MyVar1 AND 1 < MyVar2

```
