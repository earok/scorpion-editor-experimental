### SpawnOnBlock

**Category:**
Level

**Syntax:**

```scorpionengine
SpawnOnBlock ActorType X=Expression Y=Expression
```

**Description:**

Spawn an actor on a block

ActorType: The type of actor
X: The X coordinate
Y: The Y coordinate

```scorpionengine

SpawnOnBlock MyActorType X=5*MyVar Y=5*MyVar

```
