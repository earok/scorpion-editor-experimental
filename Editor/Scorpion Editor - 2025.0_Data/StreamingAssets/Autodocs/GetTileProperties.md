### GetTileProperties

**Category:**
Level

**Syntax:**

```scorpionengine
GetTileProperties VarName X=Expression Y=Expression
```

**Description:**

Get the tile properties on tile coordinates location

VarName: The variable name
X: The X coordinate
Y: The Y coordinate

```scorpionengine

GetTileProperties MyVariable X=5*MyVar Y=5*MyVar

```
