### For

**Category:**
Conditional

**Syntax:**

```scorpionengine
For VarName From=Expression To=Expression Step=Expression
```

**Description:**

Execute a variable loop

VarName: The variable name
From: From this..
To: ...To this
Step: The steps size (eg 2 means skip every other number)

```scorpionengine

For MyVariable 0 10 1

```
