### ActorMelee

**Category:**
Actor

**Syntax:**

```scorpionengine
ActorMelee X=Expression Y=Expression Width=Expression Height=Expression
```

**Description:**

Do a manually programmed melee box attack

X: The X coordinate
Y: The Y coordinate
Width: The box width
Height: The box height

```scorpionengine

ActorMelee X=5*MyVar Y=5*MyVar Width=5*MyVar Height=5*MyVar

```
