### LevelCopy

**Category:**
Level

**Syntax:**

```scorpionengine
LevelCopy Width=Expression Height=Expression X1=Expression Y1=Expression X2=Expression Y2=Expression
```

**Description:**

Duplicate a rectangular area of the level

Width: The box width
Height: The box height
X1: The first X coordinate
Y1: The first Y coordinate
X2: The second X coordinate
Y2: The second Y coordinate

```scorpionengine

LevelCopy Width=1 Height=1 X1=1 Y1=1 X2=1 Y2=1

```
