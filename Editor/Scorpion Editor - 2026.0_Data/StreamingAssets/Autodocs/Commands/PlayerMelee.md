### PlayerMelee

**Category:**
Player

**Syntax:**

```scorpionengine
PlayerMelee X=Expression Y=Expression Width=Expression Height=Expression
```

**Description:**

Do a manually programmed melee box attack

X: The X coordinate
Y: The Y coordinate
Width: The box width
Height: The box height

```scorpionengine

PlayerMelee X=1 Y=1 Width=1 Height=1

```
