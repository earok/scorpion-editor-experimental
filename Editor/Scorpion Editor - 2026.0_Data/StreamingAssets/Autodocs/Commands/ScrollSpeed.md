### ScrollSpeed

**Category:**
Display

**Syntax:**

```scorpionengine
ScrollSpeed X=Expression Y=Expression ResetMovement
```

**Description:**

Set how fast the camera auto scrolls in X and Y

X: The X coordinate
Y: The Y coordinate
ResetMovement: Resets Actor Movement to new type

```scorpionengine

ScrollSpeed X=5*MyVar Y=5*MyVar ResetMovement

```
