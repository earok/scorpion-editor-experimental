### ActorMelee

**Category:**
Actor

**Syntax:**

```scorpionengine
ActorMelee X=Expression Y=Expression Width=Expression Height=Expression
```

**Description:**

Do a manually programmed melee box attack

X: The X coordinate
Y: The Y coordinate
Width: The box width
Height: The box height

```scorpionengine

ActorMelee X=1 Y=1 Width=1 Height=1

```
