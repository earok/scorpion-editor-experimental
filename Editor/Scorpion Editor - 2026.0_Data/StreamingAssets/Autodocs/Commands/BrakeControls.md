### BrakeControls

**Category:**
Controls

**Syntax:**

```scorpionengine
BrakeControls PlayerId !Bits
```

**Description:**

Change which buttons implement brake (driving)

PlayerId: The player Id such as Player1/2/3/4 or just Player for the current active Player
!Bits: The Bits for the drive control operation - !UDLR12345678JNFH

```scorpionengine

BrakeControls Player1 !U2

```
